# Learn Rust

本项目用于学习和实践 Rust 编程语言的基础知识与进阶用法。

## 目录结构

- `src/`：源代码目录
- `Cargo.toml`：项目配置文件

## 如何运行

1. 安装 [Rust](https://www.rust-lang.org/zh-CN/tools/install)。
2. 在项目根目录下运行：

   ```bash
   cargo run
   ```
